# 📈 WealthFlow AI

**WealthFlow** is an intelligent, Python-based financial agent dashboard designed to provide users with personalized investment strategies, market explanations, and asset allocation advice. 

By leveraging the power of **Django** for a robust backend and **OpenRouter (Gemini 3.5)** for advanced AI logic, WealthFlow acts as a personal digital financial advisor.

## 🚀 Key Features
* **AI-Powered Financial Insights:** Generates tailored advice for stocks, bonds, and mutual funds based on user risk tolerance.
* **OpenRouter Integration:** Seamlessly connects to Gemini 3.5 for fast and highly accurate AI query processing.
* **Django Backend:** Built on a secure, scalable Python framework.
* **Interactive Local Dashboard:** Clean and responsive UI for users to communicate directly with the finance agent.

## 🛠️ Tech Stack
* **Backend:** Python, Django
* **Database:** SQLite
* **AI / LLM:** OpenRouter API, Gemini 3.5
* **Version Control:** Git & GitHub
